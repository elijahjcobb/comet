# comet

## Installing
From the root of this project, run `yarn install`.

## Running

### front
To run the front end, a Next.js app, run `npm run front` from the root of this project.

### api
To run the api (backend), run `npm run api` from the root of this project.


### Misc.
To run an npm for a specific workspace, append `-w <front | api>` 

For example:
* `npm run front` -> `npm run dev -w front`
